# NeuroMode

NeuroMode is a cognitive regulation engine for AI-assisted development workflows.

It measures task complexity (Cognitive Friction Score) and activates progressive task decomposition when high cognitive load is detected.

## How to Run

npm install
npm start
or
neuromode.js


## Features

- Cognitive Friction Score
- Adaptive task decomposition
- Progressive step execution
- Sequential advancement control

## Example

Input:

Develop a scalable multi-tenant SaaS platform with authentication and analytics

Output:

Cognitive Friction Score: 28  
High cognitive load detected.  
Progressive Decomposition Mode activated.
